# WikiPeek RU

**WikiPeek RU** — Tampermonkey-скрипт для быстрого предпросмотра статей Wikipedia при наведении на ссылки.

Скрипт рассчитан на русскоязычных пользователей: он сначала ищет русскую статью в Wikipedia, а если русской версии нет, показывает английскую. 
Создано ChatGPT

## Возможности

- Показывает карточку предпросмотра при наведении на ссылку
- Работает с любыми ссылками, а не только со ссылками на Wikipedia
- Сначала ищет русскую статью
- Если русской статьи нет, показывает английскую
- Поддерживает тёмную, светлую и системную тему
- Позволяет закрепить карточку средней кнопкой мыши
- В закреплённой карточке можно выделять и копировать текст
- Использует официальный Wikipedia API

## Установка

1. Установите [Tampermonkey](https://www.tampermonkey.net/)
2. Создайте новый userscript
3. Вставьте код скрипта
4. Сохраните скрипт
5. Обновите страницу и наведите курсор на любую ссылку

## Управление

| Действие | Результат |
| --- | --- |
| Навести на ссылку | Показать карточку предпросмотра |
| Нажать СКМ по ссылке | Закрепить карточку |
| Выделить текст в закреплённой карточке | Можно скопировать текст |
| `Esc` | Закрыть карточку |
| Меню Tampermonkey | Переключить тему |

## Как это работает

Если ссылка ведёт на Wikipedia, скрипт берёт название статьи напрямую.

Если ссылка обычная, скрипт формирует поисковый запрос из:

- текста ссылки
- атрибута `title`
- атрибута `aria-label`
- читаемой части URL

После этого WikiPeek RU ищет подходящую статью в Wikipedia и показывает короткую карточку предпросмотра.

## Приоритет языка

1. Русская статья в Wikipedia
2. Английская статья, если русской версии нет
